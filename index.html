Projeto - Jovem Programador

Alunos aprendem na teoria e na prática e publicam uma página na web.
No último dia 12, os professores Michael e Eduardo, ensinaram os alunos do 7ºSemestre do curso de Sistemas de Informação, a publicar um projeto web, utilizando uma conta no GitHub. Nas primeiras aulas, os professores apresentaram um sistema web, que auxilia nos primeiros passos para criação de arquivos HTML, CSS e Java Script. O sistema pode ser visualizado atravez do link https://dash.generalassemb.ly/, onde é ensinado passo a passo, como criar uma página web, fazer a formatação dos textos, criar scriprs e muito mais. Após essa iniciação, os professores deram um exercício prático, para que os alunos publicassem um blog, utilizando o conhecimento dos recursos disponibilizados pelo sistema Dash, juntamente com um sistema que faz o controde de versão e a publicação dos arquivos (GitHub) e um editor de HTML, o Sublime. Com essas ferramentas, os alunos foram capazes de construir uma página do blog, já na primeira aula após os treinamentos.
